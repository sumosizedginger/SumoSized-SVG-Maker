# SumoSized SVG Maker - Feature Catalog

SumoSized SVG Maker is a professional-grade, hardware-accelerated generative design studio. Built for designers, developers, and autonomous agents, it provides a deterministic environment for high-fidelity vector and motion art.

## 🎨 Creative Generation Suite

### 1. Generative Art Engines

- **Quantum Core**: Deep mathematical complexity for abstract structural art.
- **Sacred Geometry**: Procedural generation of Metatron’s Cubes, Mandalas, and Platonic solids.
- **Free Field Draw**: Interactive vector sketching with procedural smoothing.
- **Text Mastery**: Advanced text-on-path effects (Arch, Circle, Wave, Spiral) with perfect centering logic.

### 2. Pattern & Texture Systems

- **Flow Field**: Physics-simulated line ripples and magnetic field visualizations.
- **Dots & Grids**: Highly configurable dot matrices and grid systems with randomization.
- **Symbol Grid**: Iconographic generative grids based on deterministic symbol sets.

### 3. Layout & Branding Tools

- **Poster Generator**: Automated layout engine for typography and accent shapes.
- **Tech Badge**: Professional-grade UI asset generator for tech-labels and branding.
- **Unified Importer**: Hybrid workflow support for SVG, PNG, JPG, and WebP assets.

### 4. Elite Animation Suite (12+ Engines)

Twelve high-performance animation generators utilizing native `<animateTransform>` for 100% vector-native motion:

- **Matrix Rain**: Digital cascade with glyph randomization.
- **Quantum Pulse**: Expanding energy ripples with Zod-validated aesthetics.
- **Neural Synthesis**: Synaptic network blinking and linking animations.
- **Starfield Jump**: Hyper-speed hyperspace zoom.
- **Orbit & Kinetic Flow**: Orbital gravity simulations and fluid motion.
- **Glitch & Neon**: Digital interference and flickering light simulations.
- **Event Horizon & Chroma**: Complex distortion and color-shift animations.

## 🛠️ Technical Powerhouse

### 📹 Pro-Grade Media Pipeline

- **Hardware Acceleration**: Mediabunny engine utilizing WebCodecs API for up to **50x faster** video exports.
- **Format Support**: MP4 (H.264), WebM (VP9), MOV (ProRes-style transparency), PNG, JPEG, and raw SVG.
- **FFmpeg Fallback**: Integrated FFmpeg.wasm for environments without native hardware codecs.
- **Frame-Buffer Mastery**: SharedArrayBuffer-backed zero-latency frame capture.

### 🛡️ Security & Isolation

- **Cross-Origin Isolation (COI)**: Strictly enforced COOP/COEP local environment for secure, high-performance WASM execution.
- **100% Client-Side**: No user data, images, or videos ever leave the local browser environment.
- **Sanitized Imports**: Automatic script-stripping for imported SVG assets.

### 🚀 Performance & DX

- **Interaction to Next Paint (INP)**: Tuned for `< 16ms` render cycles via Svelte 5 Runes.
- **Headless Agent API**: Global `window.SumoSvgApp` interface for autonomous agent control and integration.
- **Deterministic Engine**: Identical seeds and parameters ALWAYS yield identical visual outputs.
- **Local Persistence**: State-aware session restoration and robust JSON-based preset management.

### 🎭 Visual Post-Processing

- **Dynamic Layer Stack**: Multi-layer composition with support for adding, reordering, and clearing via UI/API.
- **Blending & Opacity**: Per-layer control for standard SVG blend modes (Multiply, Screen, Overlay, etc.) and linear opacity.
- **Spatial Transforms**: Per-layer X/Y offsets, non-uniform scaling, and rotation.
- **Filter Registry**: Native support for complex SVG filter chains (Gaussian Blur, Displacement Maps, Turbulence).
- **Dynamic Masking**: Non-destructive alpha-masking engine allowing any layer to clip any other layer.
