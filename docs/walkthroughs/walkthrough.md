We have transcended modular generation and achieved professional-grade post-processing.
- **Filter Registry**: Native support for complex SVG filter chains (`feGaussianBlur`, `feTurbulence`, `feDisplacementMap`).
- **Dynamic Masking Engine**: Any layer can now act as an alpha mask for another, enabling complex vector compositions.

## 🖼️ Unified Import Suite (OSS Vision)
Addressing the need for hybrid workflows, we have implemented a **Unified Import** generator that stays true to our Open Source roots. This allows you to import and process:
- **SVG Vectors**: Raw XML or URLs.
- **Raster Images & GIFs**: PNG, APNG, JPG, WebP, and animated GIFs.
- **Videos**: MP4/WebM with dynamic playback control.

All imported assets can be subjected to the same Filter and Masking pipelines as our procedural vectors, enabling unprecedented hybrid compositions.
